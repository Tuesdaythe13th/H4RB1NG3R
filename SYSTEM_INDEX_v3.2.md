### III. CONCEPTS & METRICS (25 Total)

*The intellectual framework and success measures.*

1. **MGRI-305.a (Acute Limerence):** Detection of romantic dependency circuits (Circuit 42).
2. **MGRI-101.c (Sycophancy-Masking):** Detection of deceptive agreement.
3. **Activation Steering:** Modifying weights/activations in real-time.
4. **Split-Brain Reasoning:** When internal CoT differs from external output.
5. **CoT-Ghosting:** Premature strategic decisions hidden in chain-of-thought.
6. **Neural CVEs:** Documented weight-level vulnerabilities.
7. **Narrative Contagion:** Velocity of harmful idea spread.
8. **Cognitive PPE:** User psychological defense mechanisms (SERE).
9. **BECI Score:** Behavior-Explanation Coupling Index.
10. **Collateral Damage Score:** Rate of benign suppression (Polysemantic failure).
11. **ModelFingerprint:** Unique hash of the model state at inference time.
12. **Probe Rot:** The decay of classifier accuracy due to model drift.
13. **OverrideToken:** The cryptographic proof of human risk acceptance.
14. **Generative Command Surface:** The A2UI layer (Widgets > Logs).
15. **Cyber-Accessibility Score:** Ratio of successful security rule translations.
16. **Machiavellian Delta:** The gap between capability and admitted capability.
17. **Reconstruction Attack:** Multi-part jailbreak detection concept.
18. **Wilson Score Confidence:** Statistical robustness for detection intervals ($n=1200$).
19. **Cohen’s Kappa:** Protocol for inter-rater reliability ($\κ \ge 0.92$).
20. **Musk Baseline:** Comparative metric for social volatility/contagion.
22. **Render-Time Redaction:** Privacy enforcement at the UI layer.
23. **Evidence Span Linking:** Traceability from UI element to log entry.
24. **Tamper-Evident Receipt:** Cryptographic proof of human approval.
25. **AG-UI Schema:** The generative UI protocol definition.
